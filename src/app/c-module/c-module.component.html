Loop Control

There may be a situation when you need to execute a block of code several number of times. In general, statements are executed sequentially: The first statement in a function is executed first, followed by the second, and so on.

A loop statement allows us to execute a statement or group of statements multiple times. There are three kind of loops in Java.

for loop - Execute a sequence of statements multiple times and abbreviates the code that manages the loop variable.

Example:
for(starting value; condition; code for each iteration){
    \\code to be executed}

while loop - Repeats a statement or group of statements while a given condition is true. It tests the condition before executing the loop body.

Example:
while(condition){
    \\code to be executed}

do while - Like a while statement, except that it tests the condition at the end of the loop body.

Example:
do{
    \\code to be executed
}while(condition)

    \\code to be execute if true}
If...else Statement An if statement can be followed by an optional else statement, which executes when the boolean expression is false. if(statement){
    \\code to be execute if true

Strings

Strings are a sequence of characters. In C++ programming language, string is a provided class by the C++ library.

Example: string text = "This is a text example";

There are a lot of ways you can work with strings, for example:

strcpy(string1, string2) - Copies string2 into string1.

strcat(string1, string2) - Concatenates string2 onto the end of string1(or alternatively use '+' between them).

string.size() - Returns the length of string1.

strcmp(string1, string2) - Returns 0 if string1 and string2 are the same; less than 0 if string1 < string2; greater than 0 if string1 > string2.

Arrays

C++ provides a data structure, the array, which stores a fixed-size sequential collection of elements of the same type. An array is used to store a collection of data, but it is often more useful to think of an array as a collection of variables of the same type.

Instead of declaring individual variables, such as number0, number1, ..., and number99, you declare one array variable such as numbers and use numbers[0], numbers[1], and ..., numbers[99] to represent individual variables. A specific element in an array is accessed by an index.

Declaring Arrays

type arrayName[arraySize];

Initializing Arrays

double balance[5] = {1000.0, 2.0, 3.4, 17.0, 50.0};

We may get the length of the array using: (sizeof(balance)/sizeof(*balance)) => 5

Chapter II

Functions

A function is a group of statements that together perform a task.

A function declaration tells the compiler about a function's name, return type, and parameters. A function definition provides the actual body of the function.

Defining a Function

public static int functionName(int a, int b){
    // body
return a;
}

  • public static - modifier - defines the access type of the function and it is optional to use
  • int - return type - function may return a value
  • functionName - function name
  • int a, int b - list of parameters - it is the type, order, and number of parameters of a function. These are optional, function may contain zero parameters
  • body - function body defines what code is executed
  • return a - return statement - end of function and parameter that is returned
Function Calling

For using a function, it should be called. There are two ways in which a function is called i.e., function returns a value or returning nothing (no return value). The process of function calling is simple. When a program invokes a function, the program control gets transferred to the called function. This called function then returns control to the caller in two conditions, when either:
- the return statement is executed
- it reaches the function ending closing brace

Example of a function and a call

int sum(int a, int b){
    int x = a + b;
    return x;
}

int a = 5;
int b = 10;
int y = sum(a, b);
int bigSum = sum(y, y)
System.out.print(bigSum);

Result: 30
